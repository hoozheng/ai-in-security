# ai-in-security

现如今的网络安全已经不再是写一个工具，弄几条规则来和黑灰产进行对抗的时代了。由于云计算、大数据的高速发展，能够使用我们更好的利用大数据来分析网络安全行为，以数据驱动的网络安全日益显得日益紧迫，那么对数据的分析、处理、计算的工程实践就显得非常重要了，本文仅仅使用一些简单的介绍记录一下作者是如何在这条AI道路上进行探索前行的，能让更多后来者可以少走一些弯路。

一，我的机器学习启蒙：
作为一个安全工程师转向大数据挖掘领域其实有一道极其难以逾越的鸿沟就是数学基础薄弱。本书能够很好的用大白话从统计学、数据挖掘算法、实际应用案例带你入境，只有在理解了算法、数学公式背后的真实物理含义，才能够很好的进一步学习机器学习的实战。

在完成以上学习后，我们基本也对机器学习算法有了一个比较浅显的了解了，也能简单对天气预报进行一个预测、也能知道一些常用机器学习算法、10大数据挖掘算法等。
预计学习时间：30天

二、机器学习入门篇

本书比较详细的讨论了监督学习、无监督学习的基本算法，非常详细的阐述了各算法理论、实践、公式背后的意义等等，这是一本比如适合入门的机器学习书籍并且使用的是python非常适合初学者。

<div align=center><img width="200" height="250" src="https://github.com/hoozheng/ai-in-security/blob/master/res/pics/458211a6-3169-41d5-bff5-ad804f28eec3.png"/></div>

在学习本书的过程中，最好能每一行代码自己都调试通过，达到真正理解作者的意图，在调试过程中我采用的是python 3.4 + SK-Learn（下载地址：http://scikit-learn.org/）

matplotlib：matplotlib-2.1.2-cp34-cp34m-win_amd64.whl  <br/>
scipy：scipy-0.19.1-cp34-cp34m-win_amd64.whl  <br/>
whl下载地址：https://www.lfd.uci.edu/~gohlke/pythonlibs/#numpy  <br/>

预计学习时间：120天  <br/>
三、提高篇  <br/>
本书大量的关于深度学习中BP、CNN以及RNN网络等概念的数学知识解析，加以更朴素的语言与类比，使得非数学专业的程序员还是能够比较容易看懂，那些需要扩展深度学习领域的同学可以进一步读一下这本书为后面的Deep Learning打下比较好的基础。  <br/>

五、应用篇  <br/>
目前这部分还没有合适的书籍，希望我们在今年学习的过程中能够沉淀出大量的好文章能够编撰成册共大家欣赏。  <br/>
六，勿在浮沙筑高台  <br/>
数学是机器学习的重要基础，大家可以在开始学习之前先简单温习一下数学的基本知识，google看一下就好了。  <br/>
1，导数、偏导数、方向导数、梯度、梯度上升、梯度下降  <br/>
2，标量、向量、矩阵、张量的计算  <br/>
3，欧式距离、曼哈顿距离计算  <br/>
4，三角函数计算（sin、cos、tag、ctag）  <br/>
5，概率论、条件概率、贝叶斯、等基本方法  <br/>
6，二项分布、高斯分布等问题  <br/>
7，期望与方差  <br/>
8，信息、香农熵的计算 <br/> 
学习的过程有的时候是极其枯燥的，尤其是自然科学领域的学习，一定要循序渐进逐步推进，急不得也投机不得，眼高手低的话会让你学到怀疑人生，最后就真的变成从入门到放弃了。只有真正明白了数学背后的含义你才能明白算法设计的初衷。预祝每一个在AI IN Security路上摸爬滚打的人都能早日领略到算法之美。  <br/>
